# ！！！非新版日报模板，有空更新
项目名： 轻松日报 1.2【已弃用】
简介：下班写日报，修改内容、发送邮件，操作很繁琐？本程序可以自动修改要修改的Excel内容，自动发送邮件，只需简单配置后，就可以轻松使用。
适用：万维艾斯日报

1.2 更新
新增发送失败时重发机制

配置：
1、 运行环境
    Python3，pip3

2、 安装框架
    openpyxl、email、smtplib
    eg: 终端输入pip3 install openpyxl

3、获取授权码
    登录自己的邮箱网站
    开启SMPT服务
    生成授权码

4、 修改config

5、 修改“个人工作日志.xlsx”
	只需修改其中的个人姓名，保存。其他不要修改，文件名也不要修改。

注意：
①目前仅支持163和qq邮箱
② 163如果有554错误，是因为被认为是垃圾邮箱，解决方案：在抄送中加入自己的发送邮箱
